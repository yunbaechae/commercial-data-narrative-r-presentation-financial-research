# Commercial Data Narrative: R Presentation for Financial and Research Insights

STAT1378 Coding and Communication in Statistics Presentation (2021)

This presentation used R Markdown to create a visual narrative from performance and cultural data (e.g., Billboard, audio features). Focused on storytelling, trends, and commercial implications.

## Key Tasks
- Analyzed trends in music/performance data  
- Created slides with visualizations and insights  
- Recorded presentation video for clear communication  

## Skills Demonstrated
- R Markdown for presentation and reproducible slides  
- Data visualization and narrative storytelling  
- Commercial insight communication  
- Presentation recording and delivery
