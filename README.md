# Commercial Data Narrative: R Presentation for Financial and Research Insights

STAT1378 Coding and Communication in Statistics Presentation (2021)

This presentation used R Markdown to deliver a visual and narrative-driven analysis of performance and cultural data trends. Focused on storytelling, key metrics communication, and commercial implications.

## Key Tasks
- Analyzed trends in music/performance data (Billboard charts, audio features)  
- Created reproducible slides with visualizations and insights  
- Recorded video presentation for effective delivery  

## Skills Demonstrated
- R Markdown for presentation slides and reproducible narrative  
- Data visualization and trend storytelling  
- Commercial insight communication  
- Presentation recording and stakeholder delivery  

## Submission Details
- Original R Markdown source (Presentation.Rmd) included for reproducibility  
- Rendered outputs: Presentation.pdf (PDF slides), Presentation.html (interactive HTML)  
- Video recording (zoom_0.mp4) for full presentation view  
- **Note on large file**: Billboard.csv (52MB) exceeded GitHub single file upload limit (50MB) and was skipped. All other essential files (Rmd, PDF, HTML, video, images, bib) are included.  
- **vignettes/articles/** folder contains the main report-style documentation: original R Markdown source + rendered outputs, used to organize analysis code, explanations, and results.
